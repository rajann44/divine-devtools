How to Setup

Load unpacked and run Divine DevTools locally in less than a minute.

01

Download Package

Download the built ZIP folder from our GitHub Releases page or build it manually using node scripts.

02

Enable Developer Mode

Go to chrome://extensions/ in your browser and toggle the "Developer Mode" switch in the top right.

03

Load Unpacked

Click "Load unpacked" in the top-left and select the extracted folder to install the Divine DevTools.
